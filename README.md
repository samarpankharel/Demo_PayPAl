# Demo PayPal - E-Commerce T-Shirt Store

## Project Overview
This is a demonstration e-commerce platform for a premium T-shirt store that integrates the PayPal payment gateway. The project showcases a modern, responsive web interface for browsing and purchasing T-shirts online with secure payment processing through PayPal.

## Website Description
**Premium T-Shirts Store** is a full-stack e-commerce application built to:
- Showcase a curated collection of 7 different T-shirt products across various categories
- Provide a responsive and user-friendly shopping interface
- Enable secure payments through PayPal integration
- Offer a seamless customer experience from browsing to checkout

### Product Categories
- **Classic**: Timeless vintage designs
- **Streetwear**: Bold urban graphics
- **Minimal**: Clean and modern styles
- **Sports**: Performance-oriented tees
- **Oversized**: Relaxed and trendy fits
- **Premium**: Luxury cotton blends
- **Limited Edition**: Exclusive collector items

### Technical Stack (Current)
- **Frontend**: HTML5, CSS3
- **Version Control**: Git & GitHub
- **Deployment**: GitHub Repository
